# 03-MySecondJava

* Overloading 覆載

        相同function 名稱，建構多功能性的使用
        function(String)  /  function(String , int)

* Encapsulation 封裝

        定義Class，只提供必要資訊，不用知道內部運作。
        private / public
        getter() / setter() / toString()


* Inheritance 繼承

        父/子類別的繼承關係
        所有的class 都繼承Object
        Dog is animal , animal can be a kind of Dog.


* Polymorphism 多型
